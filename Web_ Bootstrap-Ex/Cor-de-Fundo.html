Esta página tem o seguinte CSS, tornando-a vermelha:

body { background-color: red; }

Porém, o código abaixo faz com que, ao redimensionar o navegador, outro CSS seja ativado sempre que o tamanho do navegador seja menor que 700px, fazendo com a página torne-se azul!

@media screen and (max-width: 700px) { body { background-color: blue; } }
